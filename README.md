This project recommends songs based on the **user's facial expression** detected via webcam in real-time.

---

## Features

- Detects user mood using **face-api.js**
- Fetches songs from **MongoDB** based on detected mood
- Upload audio files to **ImageKit**
- Responsive and modern UI using **TailwindCSS**
- Audio player integrated for each song
- Supports moods like `happy`, `sad`, `neutral`, etc.

---

## Tech Stack

- **Frontend:** React, TailwindCSS, face-api.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **File Storage:** ImageKit
